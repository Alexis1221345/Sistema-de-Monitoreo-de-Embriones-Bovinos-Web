# SMEB - Sistema de Monitoreo de Embriones Bovinos

Aplicación web estática tipo landing page para presentar el proyecto **SMEB**, un sistema de monitoreo de embriones bovinos orientado a visualización, sensibilización y explicación del flujo de datos del proceso.

## Descripción

El sitio concentra en una sola página información visual sobre:

- La problemática del monitoreo de embriones.
- La base científica del control térmico y el muestreo.
- Termografías, comparativas y flujo IoT.
- Un simulador visual para explicar el comportamiento del sistema.

## Tecnologías

- HTML5
- CSS3
- JavaScript
- Chart.js
- Google Fonts

## Estructura

- `index.html`: página principal del proyecto.

## Uso

Solo abre `index.html` en un navegador moderno para visualizar el proyecto.

## Notas

El sitio está pensado como una presentación visual del sistema y puede ampliarse fácilmente con nuevas secciones, imágenes o integración con backend.
